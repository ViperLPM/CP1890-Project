This a College project that required us to make a website using html, python, and javascript.

To use this website, heres what you need to do:
1. ensure python3 and git are installed on your computer
2. open the cmd line and enter pip install flask flask_session
3. open git and after choosing the directory you want the project to be in, enter git clone git@github.com:ViperLPM/CP1890-Project.git\
   (feel free to copy the url from here and paste it to the git cmd line with shift-ins)
4. in the folder for the project, use python to run the main.py file
5. enter the ip address on the server at port 5000 (can be any)
6. Done! Try out adding and removing cars to the session
